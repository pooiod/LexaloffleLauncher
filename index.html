Lexaloffle Launcher

An unnecessary launcher program for PICO-8, Voxatron, and other things lexaloffle.

This is an unofficial, fan-made application and is not affiliated with Lexaloffle.

About the Project

Lexaloffle Launcher is a lightweight desktop wrapper for the lexaloffle.com website. It integrates directly with your local game installations, adding quality-of-life features that I always wanted. The primary motivation for creating this was simple: I wanted a way to automatically keep my fantasy consoles updated.

Features

  • Auto-Updates: The launcher automatically checks for new versions of your registered applications.
  • Easy Config Editor: A built-in config editor allows you to view and modify configs for any of your installed apps without having to dig through your user data folders.
  • Integrated Browsing: Enjoy the full lexaloffle website, including the BBS and your account pages.
